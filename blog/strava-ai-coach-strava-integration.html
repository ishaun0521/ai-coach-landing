Strava 與 AI 教練的整合：讓數據教練你自己

這篇文章記錄我用 OpenClaw + Strava API 打造個人 AI 教練系統的完整過程。從自動抓取訓練數據、生成每週分析報告，到規劃未來四週的訓練課表——全部不需要任何付費教練服務，用數據與系統為自己建立一套完整的運動教練機制。

前言：為什麼要把 Strava 變成教練？

我從 2022 年開始認真跑步，2024 年開始騎車。用了 Strava 超過三年，它記錄了我的每一筆數據——但記錄完之後呢？

多數業餘運動者的痛點不是沒有數據，而是：

  • 每週結束後沒有系統性地回顧訓練量是否足夠
  • 不知道自己的疲勞指數是否過高、該不該休息
  • 想安排訓練計畫，但從來沒有真的安排過
  • 比賽前兩週才發現長距離課表根本還沒跑過

我需要的是一個會主動提醒我、幫我分析數據、幫我安排課表的「教練」——而且是每個人都能擁有的那種。

系統架構

整個系統由三個核心模組組成：

Strava API
運動數據來源
OpenClaw
AI 大腦與排程引擎
LINE / Telegram
教練訊息推播

Step 1：Strava API 授權設定

Strava 提供免費的 OAuth 2.0 API，只要你是運動員本人，就可以申請自己的 API Key。流程很簡單：

  1. 申請 Strava API 應用程式：strava.com/settings/api，填寫應用程式名稱與描述，Callback URL 填 http://localhost
  2. 取得 Client ID、Client Secret、Refresh Token
  3. 設定 OpenClaw Strava Skill，把憑證寫入環境變數

Step 2：自動抓取每筆活動數據

用 Strava API 的 /athlete/activities endpoint，我可以抓到所有活動：

GET https://www.strava.com/api/v3/athlete/activities
?after={timestamp}&per_page=200

回來的資料包括：距離、爬升、心率、配速、功率（騎車）。但重點不是 raw data，而是我設計的幾個關鍵指標：

  • TRIMP（訓練負荷）：根據心率區間計算單次訓練的疲勞指數
  • ATL / CTL / TSB：急性訓練負荷、慢性訓練負荷、疲勞指數——跑步圈最常用的三條線
  • 每週總里程與爬升：自動彙整，不需要打開 Strava app
  • 強度分布：Easy / Tempo / Threshold / VO2max / Rest 各佔多少 %

Step 3：教練系統的 Prompt 設計

這是核心。我的教練不是只回答「你今天跑了多少」的機器，而是要能：

你是一位專業的鐵人三項教練，服務一位業餘運動員。
運動員背景：33歲男性，月跑量 150-200km，業餘好手。
每週一早上 8:00，你要主動提供：
1. 上週訓練回顧（TRIMP、里程、爬升、強度分布）
2. 目前疲勞指數分析（ATL/CTL/TSB 三條線的狀態）
3. 本週訓練建議（哪幾天該練、練什麼、強度多少）
4. 任何需要注意的警訊（受傷風險、訓練過度）

說話風格：直接、有數據根據、不廢話。像個真正的教練。

Step 4：每月訓練課表生成

每個月第一天，我讓 AI 教練根據當月的目標比賽（大方向）和目前的體能狀態，生成一整個月的訓練規劃。

教練：「根據你的半馬 PB 2:59:02 和目前的月跑量，我建議這個月把重點放在 Zone 2 有氧基礎建設上。週末 LSD 可以拉到 20-22km，但中間配速要壓在 5:30/km 以下。」

實測結果：三週後的變化

我跑了三週，用這個系統：

+18%
平均每週里程增加
Zone 2 配速 ↓15s/km
有氧基礎更扎實
TSB 常態維持 +15~+25
比賽日不在疲勞高峰期
每週主動休息 1 天
系統自動建議，不是靠意志力

AI 教練如何幫你建立自己的運動教練系統

很多人問我：這種東西一般的業餘運動員也能做出來嗎？

答案是：當然可以。這個系統的核心價值不是用多少技術，而是讓「建立教練」這件事變得每個人都能做到。

傳統上，你需要：

  • 花錢聘請一位教練（而且好教練不便宜）
  • 自己研究訓練理論（需要大量時間與知識）
  • 靠意志力自我監督（多數人撐不過三週）

用 AI 教練系統，你可以：

  • 用你自己的數據訓練：Strava 記錄了你所有的訓練歷史，這些數據就是教練了解你的基礎
  • 建立持續的教練關係：不是一次性諮詢，而是每天、每週、每月的持續互動
  • 把專業教練的邏輯系統化：TRIMP、ATL/CTL/TSB、訓練區間……這些专业概念变成自动化的日常建議
  • 自己掌握自己的進度：不再依赖别人告诉你该怎么练，而是有系统有根据地自主训练

重點是：系統要真的運作起來，而不是停留在想了三年還沒開始。這套東西我從想到做到大概花了兩個晚上。如果你有運動習慣，強烈建議你試試看。
